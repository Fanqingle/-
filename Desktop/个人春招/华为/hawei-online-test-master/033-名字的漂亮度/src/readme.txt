给出多个名字，计算每个名字最大可能的“漂亮度”。
输入描述:

整数N，后续N个名字

N个字符串，每个表示一个名字


输出描述:

每个名称可能的最大漂亮程度

输入例子:

2 zhangsan lisi

输出例子:
