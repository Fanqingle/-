详细描述： 函数接口说明：     public String getResult(long ulDataInput)
输入参数：          long ulDataInput：输入的正整数
返回值：         String
输入描述:

输入一个long型整数
输出描述:

按照从小到大的顺序输出它的所有质数的因子，以空格隔开
输入例子:
180
输出例子:
2 2 3 3 5
